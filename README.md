- **[study-rag-server](https://github.com/LuisReinoso/study-rag-server)** `TypeScript` · self-hosted RAG over your Obsidian notes with open models only (llama.cpp / Ollama). Pluggable LLM and vector store (pgvector), pure unit-tested core.
- **[obsidian-study-rag](https://github.com/LuisReinoso/obsidian-study-rag)** `TypeScript` · the Obsidian plugin: ask your vault, answers grounded in your notes with citations.
- **[git-history-browser-extension](https://github.com/LuisReinoso/git-history-browser-extension)** `JavaScript` · Git History button for GitHub / GitLab / Bitbucket. 120+ stars, 7,000+ installs.
